部署后图片不显示

（亲测成功）解决方法

  • 修改配置文件

  • 在根目录下配置文件_config.yml 中有 post_asset_folder:false改为true。

  • 这样在建立文件时，Hexo会自动建立一个与文章同名的文件夹，这样就可以把与该文章相关的所有资源（图片）都放到那个文件夹里方便后面引用

  • 我这里放的图片是：sky2.jpg
